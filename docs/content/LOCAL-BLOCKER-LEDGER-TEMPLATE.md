# Local Blocker Ledger Template

This local blocker ledger template is repo-local, template-only, non-live, and not evidence. It is a placeholder structure for a future dry-run and must not record real blockers, actual owner notes, owner feedback, owner decisions, owner corrections, owner sign-off, acceptance results, preview evidence, production evidence, or deployment approval.

Every sample row must remain placeholder-only. Evidence status must remain `[NOT EVIDENCE / NOT RECORDED]`. Deployment approval status must remain `[DEPLOYMENT APPROVAL: NOT GRANTED]`.

## Blocker Types

- Owner input missing.
- Local correction required.
- Public visibility blocked.
- Protected admin review required.
- Fake-fact risk.
- Public leakage risk.
- Provider/runtime blocked.
- Deployment planning blocked.
- Requires separate deployment approval.
