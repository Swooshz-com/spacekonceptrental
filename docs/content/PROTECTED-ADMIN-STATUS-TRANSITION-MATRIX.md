# Protected Admin Status-Transition Matrix

This matrix is repo-local, template-only, non-live, and not evidence. It maps protected admin state changes for the current furniture/event rental website. It does not approve deployment, provider configuration, owner sign-off, preview evidence, production evidence, public quote tracking, ecommerce flows, or customer accounts.

Forbidden implications for every transition: No guaranteed availability. No confirmed booking. No public quote status tracking. No payment/order/checkout wording. No deployment approval.
